## KanbanApp Api with Node.js - Typescript - Clean architecture

#### Installation

- Clone .env.template file and rename to .env.
- Install dependencies.

```bash
  yarn i
```

- Build database

```bash
  docker-compose up -d
```

- Populate Database

```bash
 yarn populate-mongo-db
```

- Ejecutar proyecto en modo desarrollo

```bash
  yarn dev
```
